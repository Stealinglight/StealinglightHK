Output: Fonts bundled locally, CSP tightened, CDN URL configurable via env var with fallback.

**1. Update src/main.tsx** -- add Fontsource imports at the top of the file, before other imports:

```typescript
import '@fontsource-variable/inter';
import '@fontsource-variable/space-grotesk';

import './styles/index.css';
```

The Fontsource imports are side-effect imports that register @font-face declarations. Vite bundles the WOFF2 files automatically. Using JS import (not CSS @import) for better Vite tree-shaking. Per D-08, the variable font packages cover weight range 300-700 matching current usage.
