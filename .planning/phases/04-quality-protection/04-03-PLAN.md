export function Preloader({ onDismiss: _onDismiss }: PreloaderProps) {

For each of the 8 components below, apply the same pattern used in Preloader.tsx:

1. Add `useReducedMotion` to the existing `motion/react` import
2. Add `const shouldReduceMotion = useReducedMotion();` as the first line inside the component function body (or after existing hooks)
3. Guard entrance/scroll animations by conditionally passing `undefined` or empty objects when `shouldReduceMotion` is true

**IMPORTANT RULES:**

- User-initiated animations (`whileHover`, `whileTap`) are NOT guarded -- they are responses to deliberate user interaction and should always work.
- The `viewport={{ once: true }}` prop stays regardless -- it controls observation, not motion.
- AnimatePresence layout animations in Portfolio grid are NOT guarded -- they are functional UI state transitions (filter changes), not decorative entrance animations.
- Modal enter/exit animations in Portfolio are NOT guarded -- they are functional feedback for open/close actions.
- Only guard decorative entrance animations: `initial`/`animate` pairs and `initial`/`whileInView` pairs that create scroll-triggered reveals.
